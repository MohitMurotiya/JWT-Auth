# JWT-Auth
Authentication &amp; Authorization using Spring Security

#### Application Properties

```
server.port = 8080
spring.data.mongodb.uri = Database connection URI
# App Properties
cms.app.jwtSecret = secret key
cms.app.jwtExpirationMs = time in milliseconds
```
